# 家庭收支管理系统

一个基于 Python 和 Tkinter 开发的家庭收支管理系统。

## 功能特点

- 收入支出记录管理
- 数据统计和分析
- 图表可视化
- AI 智能分析
- 数据导出功能
- 美观的图形界面

## 技术栈

- Python 3.x
- Tkinter (GUI)
- SQLite3 (数据库)
- Matplotlib (图表)
- Requests (AI API)

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用说明

1. 运行程序：
```bash
python family_finance.py
```

2. 主要功能：
   - 添加收支记录
   - 编辑和删除记录
   - 查看统计信息
   - 导出数据
   - AI 智能分析
